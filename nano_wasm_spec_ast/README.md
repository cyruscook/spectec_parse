# nano_wasm_spec_ast

The SpecTec specification of the "NanoWASM" language (https://github.com/WebAssembly/spec/tree/d7b678327cd370cdbc5acfa94bd108772e2bef68/spectec/doc/example) as a Rust library.

NanoWASM is a small language to demonstrate the capabilities of SpecTec. In a similar way, this crate does not serve much practical purpose, but demonstrates the usage of the `spectec_ast` crate.
